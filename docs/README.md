## 工具分类

MCP Server 提供 95+ 个实用工具，分为 10 个类别：

1. **压缩工具** (5 个) - ZIP/TAR 压缩和解压
2. **网络工具** (15 个) - 网络搜索、页面抓取、HTML 解析
3. **文件系统** (12 个) - 文件读写、搜索、比较
4. **数据处理** (15 个) - JSON、CSV、XML、YAML、TOML 处理
5. **文本处理** (9 个) - 正则表达式、编码、文本提取
6. **系统工具** (8 个) - 系统信息、CPU/内存监控
7. **实用工具** (10 个) - UUID、哈希、日期时间、密码生成
8. **Python 开发** (8 个) - 代码执行、语法验证、AST 分析
9. **UV 包管理** (9 个) - 快速包管理、虚拟环境
10. **Pylance/Pyright** (4 个) - 类型检查、代码诊断
